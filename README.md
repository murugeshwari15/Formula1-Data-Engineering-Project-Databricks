# Formula1 Data Engineering Project

## Overview
This project demonstrates a data engineering pipeline built using Databricks and PySpark.

## Technologies Used
- Databricks
- PySpark
- Delta Lake
- Spark SQL
- Unity Catalog

## Architecture
Bronze → Silver → Gold Medallion Architecture

## Steps Implemented
1. Data Ingestion
2. Data Transformation
3. Delta Lake Processing
4. Spark SQL Analytics
5. Dashboard Visualization

## Results
Created analytics dashboards showing:
- Top drivers by points
- Constructor performance
- Championship standings
